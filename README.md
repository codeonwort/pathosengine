
## What is this
OpenGL rendering library (for 3d graphics study)

* IDE: Visual Studio 2015
* Language: C++
* GL version: OpenGL 4.3

## Libraries used
* OpenGL:  glew32, glm
* Windows: freeglut
* Image:   FreeImage
* Font:    freetype
* Loader:  tinyobjloader, assimp
