
## What is this
OpenGL rendering library (for 3d graphics study)

* IDE: Visual Studio 2017, Visual Studio Code
* Language: C++
* GL version: OpenGL 4.6 Core Profile

## Third-party Libraries
* OpenGL        : glm, gl3w
* Window system : freeglut
* Image         : freeimage
* Font          : freetype
* 3D model      : tinyobjloader, assimp
